Detailed Description

template<const int Bit = 9, const int BigIntLen = 32>
class System::BigInt< Bit, BigIntLen >

表示大型整数类 Bit：大数中一位可表示的整型数位数(>=?) BigIntLen：大数的位数(>=?)，可表示 BigIntLen * Bit 位整型数
